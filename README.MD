# 🚫 什么不能吃 (WhatNotToEat)

本项目用于系统性总结各种食品安全事件、成分造假及选购禁忌，旨在为自己家人提供一份可查证、有深度的“避坑指南”。

    我们不能决定市场上有多少坏人，但我们可以选择给家人买什么样的食物。

记录存在生产造假、虚假宣传且拒绝承担责任的企业。

## 🛠️ 维护规范

为了保证信息的严谨性，每一条记录请务必包含：

    事件名称（包含品牌与具体产品）

    信息来源（官方公告或权威媒体链接）

    核心问题（如：重金属超标、添加剂违规、成分虚假等）

    执行标准（检测依据的国标 GB）

    记录时间
