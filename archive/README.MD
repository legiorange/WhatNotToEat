# 📂 避坑档案库 (Case Archives)

    这里记录了每一宗食品安全事件的调查全过程。我们不仅记录“谁出事了”，更记录“为什么出事”以及“检测标准是什么”，确保每一份警示都有据可查。

## 📅 2026 年档案

🍚 食品安全
- [2025-3-15] 315晚会

  - 工业双氧水漂白鸡爪

        - 风险等级：🛑 严重违法（违规添加非食用物质）

        - 涉事企业：四川省蜀福香食品、重庆市曾巧食品（乖媳妇）、成都市明扬食品。

        - 关联品牌：乖媳妇、蜀福香、曾巧、川小牛、明扬川青等。

        - 摘要：生产车间脏乱不堪，污水横流。厂家为使鸡爪色泽白净，违法使用强氧化剂**工业双氧水（过氧化氢）**浸泡，长期食用可损害肝肾及口腔黏膜。

        - 源头追溯：眉山金山制药、河南亿丰电子新材料违规向食品厂供应无标签工业级双氧水。

    “泡药”增重虾仁

        - 风险等级：🛑 严重造假（磷酸盐超标、欺诈增重）

        - 涉及地区：广东湛江等地 7 家水产加工厂。

        - 涉事企业：湛江尚方舟食品、湛江中青海洋水产、湛江良基冷冻食品等。

        摘要：通过浸泡复合磷酸盐（俗称“泡药”）使虾仁吸水增重，100斤虾仁能泡出124斤；且采用“过度包冰”，导致部分产品解冻后重量缩水一半以上。

    网红锅巴

        - 涉事企业：安徽宣城南江食品（旗下品牌：宫米）。

        摘要：直播间宣传“农家土灶手工制作”，实际车间污水满地，工人将脏手套、臭鞋垫与锅巴胚料同炉烘烤。

    预制菜/肉类造假

        涉事企业：部分连锁餐饮（如西贝、老乡鸡、杨国福等部分门店）存在预制菜冒充现炒且不告知行为；刘文祥等作坊用鸭肉冒充牛羊肉制作麻辣烫食材。(PS:由于不可能去连锁餐饮店，故不记载在档案中)

## 🏷️ 风险分类标签 (System Tags)

为了提高检索效率，所有档案均标注有以下标签：

1. 按风险性质

    🛑 严重造假：蓄意替换核心成分、非法添加违禁物质、伪造生产日期。

    ⚠️ 质量缺陷：重金属/理化指标超标、生产工艺不合格。

    ℹ️ 标签误导：配料表隐瞒添加剂、虚假功效宣称、混淆品牌概念。

2. 按食品类别

    #保健品 #生鲜肉类 #母婴食品 #粮油调味 #零食饮品……

## 📝 归档操作规范

为了保持档案的严肃性，新增档案请遵循 [归档模板](../tools/template.md) 进行编写，并确保以下信息完整：

    证据链：必须附带官方检测机构的对比数据。

    企业信用：必须记录生产商与经销商的统一社会信用代码。

    闭环状态：记录事件的最终处理结果（如：罚款金额、是否吊销许可等）。
