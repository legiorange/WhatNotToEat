## 🏷️ 风险分类标签 (System Tags)

为了提高检索效率，所有档案均标注有以下标签：

1. 按风险性质

    🛑 严重造假：蓄意替换核心成分、非法添加违禁物质、伪造生产日期。

    ⚠️ 质量缺陷：重金属/理化指标超标、生产工艺不合格。

    ℹ️ 标签误导：配料表隐瞒添加剂、虚假功效宣称、混淆品牌概念。

2. 按食品类别

    #保健品 #生鲜肉类 #母婴食品 #粮油调味 #零食饮品……

## 📝 归档操作规范

为了保持档案的严肃性，新增档案请遵循 [归档模板](../tools/template.md) 进行编写，并确保以下信息完整：

    证据链：必须附带官方检测机构的对比数据。

    企业信用：必须记录生产商与经销商的统一社会信用代码。

    闭环状态：记录事件的最终处理结果（如：罚款金额、是否吊销许可等）。
